API SERVER RUNS ON PORT 6000.
MODELS: The database model contains of two tables products and carts table.
API: api contains of index.js file which accepts the request coming to the route

1) /product:returns all the currently availabel products in the database,so that it can be printed .
2) /cart:returns the items in the cart table.
